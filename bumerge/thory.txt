# What is Parallel Computing?

Parallel Computing is a technique where multiple tasks are executed simultaneously using multiple processors or threads.

Advantages:

* Faster execution
* Better CPU utilization
* Reduced computation time

OpenMP is used for parallel programming in this experiment.

---

# What is OpenMP?

OpenMP (Open Multi-Processing) is an API used for parallel programming in C/C++.

It provides:

* Multi-threading support
* Parallel loops
* Parallel sections

### Important OpenMP Directives Used

1. `#pragma omp parallel for`
2. `#pragma omp parallel sections`
3. `#pragma omp section`

# Bubble Sort

## Definition

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order.

Largest elements gradually move toward the end of the array like bubbles rising in water.

---

# Working of Bubble Sort

Steps:

1. Compare adjacent elements
2. Swap if left element is greater
3. Repeat for all elements
4. Continue until array becomes sorted

---

# Sequential Bubble Sort

Sequential Bubble Sort uses:

* Single thread
* One comparison at a time

### Time Complexity

O(n^2)

Disadvantages:

* Slow for large datasets

---

# Parallel Bubble Sort

Parallel Bubble Sort uses multiple threads for comparisons.

It works in:

1. Even Phase
2. Odd Phase

---

## Even Phase

Compares:

* (0,1)
* (2,3)
* (4,5)

simultaneously.

---

## Odd Phase

Compares:

* (1,2)
* (3,4)
* (5,6)

simultaneously.

---

# OpenMP Parallel Loop

```cpp
#pragma omp parallel for
```

This directive distributes loop iterations among multiple threads.

Advantages:

* Faster sorting
* Better CPU utilization

---

# Merge Sort

## Definition

Merge Sort is a Divide and Conquer algorithm.

It:

1. Divides array into halves
2. Sorts each half recursively
3. Merges sorted halves

---

# Merge Operation

The merge function combines two sorted arrays into one sorted array.

### Merge Process

1. Compare elements from both halves
2. Store smaller element first
3. Copy remaining elements

---

# Sequential Merge Sort

Sequential Merge Sort performs recursive sorting using one thread.

### Time Complexity

O(n\log n)

Advantages:

* Faster than Bubble Sort
* Efficient for large datasets

---

# Parallel Merge Sort

Parallel Merge Sort uses multiple threads to sort left and right halves simultaneously.

### OpenMP Parallel Sections

```cpp
#pragma omp parallel sections
```

Different sections run concurrently.

---

# OpenMP Section Directive

```cpp
#pragma omp section
```

Each section is assigned to different threads.

Advantages:

* Faster recursive processing
* Improved performance

---

# Recursion in Merge Sort

Merge Sort uses recursion to divide the array repeatedly until:

* Single element subarrays are obtained.

Then arrays are merged back in sorted order.
