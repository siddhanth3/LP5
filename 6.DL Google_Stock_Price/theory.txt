# Theory: Google Stock Price Prediction using Deep Learning (LSTM)

## Aim

To implement a Deep Learning model using Long Short-Term Memory (LSTM) networks for predicting Google stock prices based on historical stock market data.

---

# Introduction

Stock Price Prediction is the process of forecasting future stock prices using historical market data.

Deep Learning techniques, especially Recurrent Neural Networks (RNN) and Long Short-Term Memory (LSTM), are widely used for time-series forecasting because they can learn patterns from sequential data.

In this practical, an LSTM model is used to predict Google stock prices.

---

# What is Deep Learning?

Deep Learning is a subset of Machine Learning that uses neural networks with multiple layers to learn patterns automatically from large datasets.

Deep Learning is highly useful in:

* Stock market prediction
* Image recognition
* Speech recognition
* NLP applications

---

# What is Time Series Data?

Time series data is a sequence of data collected over time intervals.

Examples:

* Stock prices
* Weather data
* Sales records

In this experiment:

* Historical Google stock prices are used as time-series data.

---

# What is LSTM?

LSTM (Long Short-Term Memory) is a special type of Recurrent Neural Network (RNN) designed for sequential and time-series data.

LSTM can remember important information for long periods and solve the vanishing gradient problem present in traditional RNNs.

Advantages of LSTM:

1. Handles sequential data efficiently
2. Remembers long-term dependencies
3. Gives better prediction accuracy

# LSTM Neural Network Architecture

## 1. LSTM Layer

The LSTM layer learns patterns and dependencies from stock price sequences.

### LSTM Memory Concept

LSTM uses:

* Input Gate
* Forget Gate
* Output Gate

to manage information flow.

---

## Forget Gate Formula

f_t=\sigma(W_f[h_{t-1},x_t]+b_f)

The forget gate decides which information should be removed.

---

## Input Gate Formula

i_t=\sigma(W_i[h_{t-1},x_t]+b_i)

The input gate decides which new information should be stored.

---

## Output Gate Formula

o_t=\sigma(W_o[h_{t-1},x_t]+b_o)

The output gate determines the final output.

---

# Dropout Layer

Dropout is used to reduce overfitting.

It randomly disables neurons during training.

Advantages:

* Better generalization
* Prevents memorization

# Mean Squared Error (MSE)

MSE measures the average squared difference between actual and predicted values.

### Formula

MSE=\frac{1}{n}\sum_{i=1}^{n}(y_i-\hat{y_i})^2

Lower MSE indicates better prediction accuracy.
