# Theory: Plant Disease Detection using Deep Learning (CNN)

## Aim

To implement a Deep Learning model using Convolutional Neural Network (CNN) for detecting plant diseases from leaf images.

---

# Introduction

Plant diseases reduce crop quality and agricultural productivity. Manual disease detection is time-consuming and requires expert knowledge.

Deep Learning techniques, especially Convolutional Neural Networks (CNNs), are widely used for automatic image classification and disease detection.

In this practical, a CNN model is trained to identify diseases from plant leaf images.

---

# What is Deep Learning?

Deep Learning is a subset of Machine Learning that uses neural networks with multiple layers to learn patterns automatically from data.

It is highly effective in:

* Image Recognition
* Speech Recognition
* Natural Language Processing
* Disease Detection

---

# What is CNN?

Convolutional Neural Network (CNN) is a special type of neural network mainly used for image processing tasks.

CNN automatically extracts:

* Edges
* Shapes
* Patterns
* Texture features

from images.

---

# Dataset Information

The dataset contains images of:

* Healthy plant leaves
* Diseased plant leaves

Different disease categories are stored in separate folders.

The CNN model learns features from these images and predicts the disease class.

# Image Preprocessing

## Image Resizing

All images are resized into fixed dimensions so that the CNN model can process them efficiently.

Example:

* 128 × 128 pixels

---

## Image Normalization

Pixel values are scaled between 0 and 1.

### Normalization Formula

x_{norm}=\frac{x}{255}

Normalization improves training performance and convergence speed.

---

# Data Augmentation

Data augmentation increases dataset diversity by applying transformations such as:

* Rotation
* Flipping
* Zooming
* Shifting

Advantages:

* Reduces overfitting
* Improves model generalization

---

# CNN Architecture

## 1. Convolution Layer

The convolution layer extracts important image features such as edges and textures.

### Convolution Operation

(I*K)(x,y)=\sum_m\sum_n I(m,n)K(x-m,y-n)

Where:

* I = Input Image
* K = Kernel/Filter

---

## 2. ReLU Activation Function

ReLU introduces non-linearity into the network.

### Formula

f(x)=\max(0,x)

Advantages:

* Faster training
* Solves vanishing gradient problem

---

## 3. Pooling Layer

Pooling reduces image dimensions and computational complexity.

### Max Pooling

Selects the maximum value from a region.

Advantages:

* Reduces overfitting
* Improves feature extraction

---

## 4. Flatten Layer

The Flatten layer converts 2D feature maps into a 1D vector before passing data to dense layers.

---

## 5. Dense Layer

Dense layers perform classification based on extracted features.

---

## 6. Output Layer

The output layer predicts the disease category.

For multi-class classification:

* Softmax activation is used.

### Softmax Formula

Softmax(x_i)=\frac{e^{x_i}}{\sum_{j}e^{x_j}}

Softmax converts outputs into probability values.

---

# Model Compilation

The CNN model is compiled using:

* Optimizer: Adam
* Loss Function: Categorical Crossentropy
* Metric: Accuracy

---

# Categorical Crossentropy Loss

Used for multi-class classification problems.

### Formula

L=-\sum_{i=1}^{C}y_i\log(\hat{y_i})

Where:

* yᵢ = Actual value
* ŷᵢ = Predicted probability

Lower loss indicates better model performance.

---

# Model Training

The model is trained using:

* Epochs
* Batch Size

### Epoch

One complete pass of the training dataset through the neural network.

During training:

* CNN learns image patterns
* Updates weights
* Improves prediction accuracy

---

# Model Evaluation

After training, the model is tested on unseen images.

Performance metrics:

* Accuracy
* Loss

Higher accuracy indicates better disease prediction capability.

# Prediction Process

The trained CNN model takes a leaf image as input and predicts:
